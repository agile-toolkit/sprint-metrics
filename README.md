# Sprint Metrics

A sprint metrics dashboard for Scrum teams — velocity charts, burn-down/burn-up, release forecasts, and XLSX import from BacklogManager templates.

Part of the [Agile Tools](https://github.com/bthos) suite built on Management 3.0 and ICAgile source materials.

## Stack
React 18 · TypeScript · Vite · Tailwind CSS · Recharts · react-i18next (EN/RU)

## Development
```bash
npm install
npm run dev
```

## Deploy
GitHub Pages via GitHub Actions on push to `main`.
